# Architecture

## Цель
Система для детекции и трекинга автомобилей на видео/городских потоках с последующей аналитикой.

## Поток данных
1. Получение видео или RTSP потока.
2. Извлечение кадров.
3. Детекция автомобилей.
4. Трекинг между кадрами.
5. Вычисление событий и статистики.
6. Сохранение результатов.
7. Выдача через API.
